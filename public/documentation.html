Documentation Page

This is a API that provides users with access to a movie database where they could search about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Query URL Method Query parameters Request Response
Return a list of ALL movies to the user "/movies" GET "None" "None" "A JSON object holding data about all movies"
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user "/movies/:title" GET "Title" "None" "A JSON object holding all data about the searched movie"
Return data about a genre (description) by name/title (e.g., “Thriller”) "/movies/genre/:genreName" GET "genreName" "None" "A JSON object holding all infos about the genre"
Return data about a director (bio, birth year, death year) by name "/movies/directors/:directorName" GET "directorName" "None" "A JSON object holding all data about the searched director"
Allow new users to register "/users" POST "None" "A JSON object holding data about the user to add. Structured like: { name: "Lulu", favoriteMovies: [], } " "A JSON object holding data about the user that was added, including an ID: { id: 1, name: "Lulu", favoriteMovies: [], } "
Allow users to update their user info (username) "/users/:id" PUT "id" "None" "A JSON object holding data about the user that was updated"
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) "/users/:id/:movieTitle" POST "id, movieTitle" "None" "A text message indicating whether the movie was seccessfully added"
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) "/users/:id/:movieTitle" DELETE "id, movieTitle" "None" "A text message indicating whether the movie was seccessfully removed"
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) "/users/:id" DELETE "id" "None" "A text message indicating whether the user was successfully removed"
