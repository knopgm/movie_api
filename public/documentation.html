Documentation Page

This is a API that provides users with access to a movie database where they could search about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Query URL Method Query parameters Request Response
Return a list of ALL movies to the user "/movies" GET "None" "None" "A JSON object holding data about all movies"
Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user "/movies/:Title" GET "Title" "None" "A JSON object holding all data about the searched movie"
Return data about a genre (description) by name/title (e.g., “Thriller”) "/genre/:name" GET "name" "None" "A JSON object holding all infos about the genre"
Return data about a director (bio, birth year, death year) by name "/directors/:name" GET "name" "None" "A JSON object holding all data about the searched director"
Allow new users to register "/users" POST "None" "A JSON object holding data about the user to add. Structured like: { Username: String, Password: String, Email: String, Birthday: Date }" "A JSON object holding data about the user that was added, including an ID. Ex.:{ ID: 1, Username: "Lulu", Password: 12344, Email: lulu@lulu.com, Birthday: 1999-04-05 }"
Allow users to update their user info (username) "/users/:Username" PUT "Username" "None" "A JSON object holding data about the user that was updated. Ex.:{ Email: lulu_Updated@lulu.com}"
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) "/users/:Username/movies/:MovieID" POST "Username, MovieID" "None" "A JSON object holding all data about the user, including the added movie. Ex.:{ ID: 1, Username: "Lulu", Password: 12344, Email: lulu@lulu.com, Birthday: 1999-04-05, FavoriteMovies: ["62711f4dfeca33c02c3251ed" ] }"
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) "/users/:Username/movies/:MovieID" DELETE "Username, MovieID" "None" "A JSON object holding all data about the user, excluding the added movie. Ex.:{ ID: 1, Username: "Lulu", Password: 12344, Email: lulu@lulu.com, Birthday: 1999-04-05, FavoriteMovies: [] }"
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) "/users/:Username" DELETE "Username" "None" "A text message indicating whether the user was deleted"
